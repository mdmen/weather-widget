# Current weather widget

Built using:
- React
- Flow
- Bootstrap
- OpenWeather
- Jest
- React Testing Library

## Usage

```html
<body>
  ...
  <weather-widget appId="<YOUR_OPENWEATHER_PUBLIC_API_KEY>"></weather-widget>
  <script async src="<PATH_TO_THE_SCRIPT>/widget.min.js"></script>
  ...
</body>
```
